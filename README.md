# 微信小程序-课程答疑系统
在传统高校课堂中，学生课后遇到问题时，内向者不好意思当面提问，班级群提问易被刷走，而反复询问相同问题也大量消耗老师精力，答疑效率低下。为此，我们开发一款课程答疑微信小程序，通过微信将提问、回答与知识管理整合为轻量级闭环。学生可随时提问（支持文字和图片），老师或助教在问题看板中统一处理，系统自动沉淀历史问答，形成可检索、复用的课程知识库，既降低提问门槛，又减少老师重复劳动。项目覆盖学生、教师和管理员三种角色，采用微信小程序原生框架搭配Java Spring Boot后端，通过WebSocket实现消息实时推送。同时引入AI智能问答，辅助老师快速响应常见问题，提升教学互动效率。我们希望让每一个问题都被看见、被回应、被记住，让每门课程生长出属于自己的活跃答疑知识库。
